## Proyecto GUI App realizada con MAUI la cual consume la API para mostrar graficas
* .Net 7
* Domain
* Microcharts.Maui 1.0.0
## Configuración
### App.xaml
```xaml
  <x:String x:Key="UrlBase">http://apifx.somee.com</x:String>
  <x:String x:Key="User">/api/user</x:String>
```
